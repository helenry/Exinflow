# Exinflow

How to run the server:
1. Start Apache and MySQL server
2. Go to server directory
3. Run `npm install` to install all dependencies
4. Run `npm run server`

How to use the Swagger documentation UI:
1. Go to `/api/docs`
2. Scroll down to User API
3. Sign up using the `/api/signup` endpoint
4. Log in using the `/api/login` endpoint. It will return a JWT
5. Copy the JWT
6. Scroll up and click the Authorize button
7. Paste the JWT
8. Click Authorize
9. You can use all of the endpoints now
